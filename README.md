# Vapor Blog
This is an Template for using vapor and material-kit

## Features

- Authentication
- Markdown support and editor
- Syntax Highlighting
- Dashboard (ing..)

## Requirements
- Vapor 3.0+
- Swift 4.1+
- PostgreSql 10.5-1.pgdg90+1
